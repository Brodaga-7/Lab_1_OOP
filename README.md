Лабораторна робота №1:SmartWatch

Функціонал програми
1.  Додавання об'єктів:
    Введення даних про модель, ціну, заряд, ОС, водостійкість та дату виготовлення.
    Валідація введених даних (наприклад, ціна не може бути від'ємною, назва не довша за 20 символів).
2.  Перегляд списку:
    Виведення всіх годинників у вигляді форматованої таблиці.
3.  Пошук:
    Пошук за назвою, точною ціною, рівнем заряду, типом ОС та водостійкістю.
    Підказки існуючих значень перед пошуком.
4.  Демонстрація поведінки:
    Імітація дзвінка, встановлення додатків та зарядки пристрою.
5.  Видалення:
    Видалення конкретного запису або групи записів за критеріями.

Мова:C#
Середовище:Visual Studio Codу

Структура класу SmartWatch
Клас містить наступні поля:
* ModelName (string) - назва моделі.
* Price (double) - ціна.
* BatteryLevel (int) - рівень заряду (0-100%).
* ManufactureDate (DateTime) - дата виготовлення.
* OsType (Enum) - тип операційної системи.
* IsWaterproof (bool) - наявність водозахисту.
